The packages.go and packages_windows.go file is generated by the scriggo command, DO NOT alter it.

# Building

To regenerate the packages file run the following:

```sh
rm -f packages.go packages_windows.go
go run github.com/open2b/scriggo/cmd/scriggo@v0.60.0 import -f packages.Scriggofile -o packages.go
go run github.com/open2b/scriggo/cmd/scriggo@v0.60.0 import -f packages_windows.Scriggofile -o packages_windows.go
```
